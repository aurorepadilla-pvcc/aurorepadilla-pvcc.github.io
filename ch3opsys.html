Operating Systems Terminology
TERM DEFINITION
Operating System The software that controls the overall operation of a computer which provides the means by which a user can store and retrieve files, the interface used to request the execution of programs, and the environment necessary to execute the programs requested.
Job Queue A storage organization in which jobs are ordered in first-in, first-out (FIFO) fashion.
FIFO The structure of a queue with a "first-in, first out" fashion.
Interactive Processing Feature developed that allowed a program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing Operating systems that provide service to mutltiple users at the same time.
Multiprogramming Technique in which time is divided into intervals and then the execution of each jobs is restricted to only one interval at a time.
Application Software Consists of the programs for perfoming tasks particular to the machine's utilization.
System Software Perfoms tasks that are common to computer systems in general.
Utility Software Consists of programs for perfoming activities that are fundemental to computer instalations but not included in the operating system.
Graphical User Interface The portion of an operating system that handles communication with users through icons (objects represented pictorially).
Shell Older user interfaces that communicated with users through textual messages using a keyboard and monintor screen.
Kernel The internal part of an operating system that contains the components that perfom the very basic functions required by the computer installation.
Device Driver The software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager Coordinates the use of the machine's mass storage facilities.
Memory Manager Charged with the task of coordinating the machine's use of main memory.
Virtual Memory Large "fictional" memory space created by paging.
Boot Strapping The procedure that transfers the operating system from mass storage to main memory.
ROM A small portion of a computer's main memory contructed from special nonvolatile memory cells.
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling Holding data for output at a later but more convenient time.
Super User Person who gains highly privileged access to the operating system to protect the computer's resources from access by unauthorized personnel.
Multi-Factor Authentication The requirement of multiple methods of authentication to verify a user's identity.
