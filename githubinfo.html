All About Github

QUESTION ANSWER
What is a version control system? A version control system is a software tool that help software teams manage changes to source code over time.
What is Git? Git is a distributed version control system that is widely used in software development and allows multiple developers to work on a project stimultaneously while tracking changes made to the codebase and enabling collaboration.
What is GitHub? GitHub is a web-based platform that provides hosting for Git repositories, it offers a collaborative environment for developers to store, manage, and share their code repositories.
What is the difference between Git and GitHub? Git is a distributed version control system while GitHub is a web-based platform that provides hosting for Git repositories.
Who started GitHub and how was it started? GitHub was created by Tom Preston-Werner, Chirs Wanstrath, and PJ Hyett in April, 2008. It was started because of the founders' ideas to create a platform that would simplify the process of collaborating on software development projects using Git.
What company owns it now? As of 2018, Microsoft own GitHub.
How much does a GitHub account cost? As of April 14, 2020, all the core GitHub features are free for everyone.
What is the Octocat? The Octrocat is the official mascot of GitHub
What is a repository? A GitHub repository is a location where developers can store, manage, and collaborate on code.
What does commit mean? Commit refers to saving changes made to a Git repository.
What does fork mean? Fork refers to the process of creating a copy of a repository from one usre's account to another user's account.
What does push mean? Push is the action of sending local changers from a developer's computer to a remote respoditory hosted on GitHub.
What are pull resquets? A pull request is a feature that allows developers to propose changes to a repository hosted on GitHub.
What are workflows? Workflows are the processes and practices that developers follow when using Git for version control.
What are issues? Issues refer to problems, bugs, or tasks that need to be addressed within a project.
What is a raw button? The raw button allows users to view the raw content of a file in a repository.
What is a blame button? The blame button allows users to see who last modified each line of code in a specific file.
