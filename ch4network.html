Networking & The Internet
TERM DEFINITION
Network Classifications
  • LAN
  • MAN
  • WAN
 A computer network is often classified as being either:
  • A local area network, which normally consists of a collection of computers in a single building or building complex.
  • A metropolitan area network which is a network of intermediate size, such as one spanning a local community.
  • A wide area network, which links machines over a greater distance, perhaps in neighboring cities or on opposite sides of the world.
Network Models
  • Bus Network
  • Star Network
 Network Models refer to classifying networks based on the topology of the network, two of the more popular topologies are:
  • Bus, in which the machines are all connected to a common communication line called a bus.
  • Star, in which one machine serves as a central focal point to which all the others are connected.
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
 Protocols are rules established by which activites are conducted.
  • CSMA/CD dictates that each message be broadcast to all the machines on the bus.
  • CSMA/CA refers to wireless networks adopting the policy of trying to avoid colisions rather than trying to detect them.
Combining Networks
  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
 Sometimes it is necessary to connect existing networks to form an extended communication system.
  • The repeater is little more than a device that passes signals back and forth between the two original buses without considering the meaning of the signals.
  • A bridge is similar to, but more complex than a repeater, it connects two buses but it does not necessarily pass all messages across the connection.
  • A switch is essentially a bridge with multiple connections, allowing it to connect several buses rather than just two.
  • Routers are special purpose computers used for fowarding messages
  • A gateway is the "point" at which one network is linked to an internet.
Client/Server Model
  • P2P: peer-to-peer
  • Distributed System
  • Cloud Computing
 The client/server model is a popular convention used for interprocess communication.
  • P2P involves processes that provide services to and receive service from each other.
  • Distributed systems consist of software units that execute as processes on different computers
  • Cloud computing, whereby huge pools of shared computers on the network can be allocated for use by clients as needed, is the latest trend in distributed systems.
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
  • Tier 1 consists of very high-speed, high-capacity, international WANs.
  • Tier 2 tend to be more regional in scope and less potent in their capabilities.
  • Tier 3 keeps the network connected even if very important tier-1 routers fail.
IP Address
  • ICANN
  • DNS
  • TLD: Top-level domain
  • Country-code TLDs
A unique identifying address assigned to each computer in a system.
  • ICANN: Internet Corporation for Assigned Names and Numbers, a non-profit organization established to coordinate the Internet's operation.
  • Domain Name System: name servers used as an internet-wide directory system.
  • Top-Level Domain consists of the suffixes used to indicate the domain's classification.
  • Country-code TLDs refer to two-letter TLDs for each specific country.
Internet Protocols
  • FTP
  • HTTP
  • SMTP
  • File Transfer Protocol: an application for accessing another computer from a great distance.
  • Hyper Text Transfer Protocol
  • Simple Mail Transfer Protocol: defines a way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web
  • URL
  • Client-side Activities
  • Server-side Activities
  • Uniform Resource Locator: the unique address given to each document.
  • Activities if they are perfomed by a client (such as a browser).
  • Activities if they are performed by a server (such as a webserver).
Internet four-layer software
  • Application Layer
  • Transport Layer
  • Network Layer
  • Link Layer
 The internet software has four layers, each consisting of a collection of software routines rather than people and businesses.
  • The application layer consists of those software unites such as clients and servers that use Internet communication to carry out their tasks.
  • The transport layer is used to send and receive messages over the Internet in much the same way that you would use a shipping company to send and receive packages.
  • It is the network layer's job to decide in which direction a packet should be sent at each step along the packet's path through the inernet.
  • The link layer has the responsibility of trasnferring the packet. Thus, the link layer must deal with the communication details particular to the individual network in which the computer resides.
Internet is:
  • Packet-switching Network
  • Fault Tolerant
  • Redundant
  • The internet is a packet-switched system through which digital data is sent by breaking the data into blocks of bits called packets, which contain both the data being transmitted and control information for routing the data.
  • Enables a computer to respond to a failure in hardware or software.
  • The redundancy of routing between two points on the internet increases the reliability of the internet and helps it scale to more devices and more people.
Cybersecurity
  • Malware
  • Virus
  • Worm
  • Trojan Horse
  • Spyware
  • phishing
  • DDoS Attack
 Cybersecurity is the study of protecting computers, networks, and data from attacks.
  • Malware refers to the use of malicious software to attack.
  • A virus is software that infects a computer by inserting itself into programs that already reside in the machine.
  • A worm is an autonomous program that transfers itself through a network, taking up residence in computers and fowarding copies of itself to other computers.
  • A trojan horse is a program that enters a computer system disguised as a desirable program, such as game or useful utility package, that it willingly imported by the victim. Once in the computer, it might have harmful effects.
  • Spyware is another form of malicious software which collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
  • Phishing is a technique of obtaining information explicitly by simply asking for it.
  • Denial of service attack is the process of overloading a computer with messages.
Protection
  • Firewall
  • Spam Filter
  • Proxy Server
  • Antivirus Software
  • Cryptography
  • Public Key Encryption
  • A firewall is a program which filters traffic passing through a point in the network.
  • Spam filters are firewalls designed to block unwanted email.
  • A proxy server is a software unit that acts as an intermediary between a client and a server with the goal of sheilding the client from adverse actions of the server.
  • Antivirus software is used to detect and remove the presence of known viruses and other infections.
  • Cryptography is the study of of sending and receiving messages securely in the presence of adversaries.
  • Public-key encryption involves techniques by which the encryption and decryption algorithms are known, and one the keys is known, but the encoded message remains secure.
